# Deep-Learning-Based-UAV-Route-Planning-Algorithm
Deep Learning-Based UAV Route Planning Algorithm
This project implements automatic inspection of wind power generation unmanned aerial vehicles and utilizes 2D images to implement automatic route planning algorithms for unmanned aerial vehicles

# 项目概况
- 本项目实现风力发电无人机自动巡检，利用2D图像实现无人机自动航线规划算法

# 算法原理
- 利用2张2D图像，实现无人机飞行航迹自动规划算法，并且实现可控的等时间、等间距拍照。实现360度无死角拍摄，全程无需人工参与。  
- Using two 2D images, implement an automatic flight path planning algorithm for unmanned aerial vehicles, and achieve controllable equal time and equidistant photography. No manual involvement is required throughout the entire process.
